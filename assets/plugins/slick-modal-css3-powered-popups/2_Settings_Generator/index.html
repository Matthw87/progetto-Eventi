Settings Generator BETA

Select the settings and when finished, copy the generated snippet to your page. Install steps »

1. Upload the plugin files (slickmodal.min.css and jquery.slickmodal.min.js) from /src/plugin folder to your web server.
2. Copy the generated snippet to your website right before closing the </body> tag.
3. Edit the content of the popup wrapper and insert your inner HTML elements

All settings have a default value defined within the plugin core. They will disappear from the generated code if the default value is selected.

For help you can also check the input placeholders which show an example or hover over a setting field to see the tip.

CSS must be properly formatted - please see the placeholder structure for detailed example.

You can always additionally edit all the values when pasting the code to your page and add/remove options manually.

Only popup types, auto close, popup/overlay/content effects and their CSS can be previewed.

As the generator is currently in BETA state please report any bugs you encounter during the usage.
